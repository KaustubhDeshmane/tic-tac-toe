# 🎮 Tic Tac Toe

A modern and responsive **Tic Tac Toe** game built using **HTML, CSS, and JavaScript**. The project features a clean glassmorphism-inspired interface, smooth animations, a live scoreboard, winner detection, draw detection, and a fully responsive design for desktop and mobile devices.

---

## 🌐 Live Demo

```text
https://KaustubhDeshmane.github.io/Tic-Tac-Toe/
```

---

## ✨ Features

* 🎮 Classic Tic Tac Toe gameplay
* 👥 Two-player mode
* 🏆 Automatic winner detection
* 🤝 Draw detection
* 📊 Live scoreboard
* 🔄 Reset board anytime
* 🎉 Winner popup animation
* ✨ Winning box highlight
* 📱 Fully responsive layout
* 🎨 Modern Glassmorphism UI
* ⚡ Smooth hover and transition effects
* 🧹 Clean and modular JavaScript code

---

## 🛠 Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/KaustubhDeshmane/Tic-Tac-Toe.git
```

### Run the Project

Open `index.html` directly in your browser, or launch the project using the VS Code **Live Server** extension.
